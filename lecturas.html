Que son las Bases de datos

El concepto de base de datos relacional o mejor dicho lo que entendemos como base de datos relacional es aquella que representa los datos y las relaciones entre los datos mediante una coleccion de tablas, cada una con un nombre unico donde una fila de una tabla representa una relacion entre un conjunto de valores. El modelo de base de datos relacional consta de 3 aspectos ) estructura de datos , integridad de datos y manipulacion de datos=

Funciones y sublenguajes (DDL y DML)

Un Sistema Gestor de Bases de Datos es un conjunto de programas de proposito general, que facilita la definicion, construccion y manipulacion de las bases de datos . Los SGBD relacionales incluyen dos tipos de lenguajes : Lenguaje de definicion de datos (DDL), este ayuda a definir la estructura y los componentes tablas, atributos y restricciones de las bases de datos. Lenguaje de manipulacion de datos (DML), Lenguaje que ayudara a la manipulacion de los datos, pudiendo ser utilizado para realizar consultas y modificaciones

Clasificacion de los tipos de datos segun nivel de abstraccion

Una base de datos con buena arquitectura debe permitir su utilizacion en distintas maquinas con distintos sistemas operativos (portabilidad). Otro de los objetivos es la abstraccion de datos, lo que significa dar a los usuarios una vision abstracta de los datos, es decir, una visualizacion de los datos , pero no el conocimiento de la estructura interna.

Niveles de abstraccion

Los niveles de abstraccion que ilustraremos seran 3 (Nivel fisico, nivel logico y nivel de vision) En el nivel fisico encontramos que es el nicel mas bajo y exhaustivo. trata los detalles del almacenamiento real. El nivel logico, describe la informacion guardada y relaciones de su almacenamiento. El nivel de vision es el nivel de abstraccion mas alto describe solo una parte de la base de datos puesto que esta es la que se le muestra al usuario.
